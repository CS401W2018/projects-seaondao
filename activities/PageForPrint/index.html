  1. Why is the viewport meta tag important?

    So that it will be responsibe to mobile devices.

  2. How can you adjust the CSS for printing?

    Create a new css for printing. Add css on html with media="print" whitch will be only loaded when ptinting.

  3. When would you want to adjust the CSS for printing?

    When you know the page is designed to be printed.

  4. Why do we need a viewport meta tag?

    This will set that width = device width and when page is loaded zoom is normal 1.0.

  5. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

    The one selected after will be appied. In this case "flex-direction: row;" will be applied.

  6. How do you change the CSS for a web page displayed on a phone?

    add "@media only screen AND (max-width: 600px)" on CSS and any device under 600px, you can change the css for those small devices there.
