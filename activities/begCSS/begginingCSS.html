Engagement - Beginning CSS

  1. Explain the difference between inline, embedded, and external CSS.

    inline - Usually done by style attribute. This is CSS that you write inside html tag.

    embedded - putting style tag on top of the page(in head tag). This will need to use CSS selector.

    external - making filename.css file just for styling.

  2. Explain why you should only use external CSS.

    Separetion of concern - by descrbing it will help SEO. It will be easier to separate task so that they can do what they are good at. It can also apply to multiple html file.

  3. What is a selector used for?

    To select whcih part do you want to apply the CSS.
